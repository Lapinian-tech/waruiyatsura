事件・公開情報ディレクトリ 一式
================================

【ファイル構成】
  admin.html                 … 管理画面（ローカルPCで開いて使うツール）
  public_site/               … 公開サイトの参考一式
    ├ index.html
    ├ styles.css
    ├ script.js
    └ data.js                … サンプルデータ入り（admin の ZIP 出力で置き換わります）

【使い方】
  1. admin.html をローカルPCのブラウザで開く
  2. 情報を入力して「保存する」（データはブラウザ内に保持されます）
     ・メイン画像：1枚（一覧・詳細の主画像）
     ・追加画像：複数登録可（詳細画面でギャラリー表示）
  3. 「公開ページを確認」で出力イメージをチェック
  4. 「公開用サイト ZIP を出力」→ 解凍
  5. index.html / styles.css / script.js / data.js を
     CoreServer 等へ FTP アップロード

【注意】
  ・admin.html は公開サーバーに置かないでください（ローカル利用専用）
  ・ZIP 出力には JSZip(CDN) を使うため、admin.html を開く端末はネット接続が必要です
  ・掲載は公式公開情報に限定し、断定表現・私刑目的・噂情報は避けてください
  ・氏名表記は「犯人／犯罪者」と断定せず「被疑者／容疑者」を用いてください
